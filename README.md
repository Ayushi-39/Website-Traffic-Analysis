## 🚀 How to Run in VS Code

### Option 1 — Live Server (Recommended)
1. Open this folder in VS Code: `File → Open Folder → traffic-analytics`
2. Install the **Live Server** extension (by Ritwick Dey) from the Extensions panel
3. Right-click `index.html` → **"Open with Live Server"**
4. Dashboard opens at `http://127.0.0.1:5500`

### Option 2 — Direct Browser
1. Double-click `index.html` — opens in your default browser
2. ⚠️ Some browsers block local font loading; Live Server is preferred

## ✨ Features

| Feature | Description |
|---------|-------------|
| **5 Dashboard Sections** | Overview, Traffic, Pages, Behavior, Recommendations |
| **10+ Interactive Charts** | Line, bar, donut, pie, bubble, horizontal bar |
| **Date Range Filter** | Toggle between 7, 14, 30-day views — KPIs update live |
| **Light/Dark Theme** | Toggle anytime — persists via localStorage |
| **Pages Table** | Sortable by any column, live search filter |
| **Export CSV** | Downloads all pages data as a .csv file |
| **Export PDF** | Ctrl+P / Print button for PDF report |
| **Conversion Funnel** | Animated drop-off visualization |
| **Hourly Heatmap** | Traffic intensity per hour of day |
| **Priority Matrix** | Impact vs Effort bubble chart for recommendations |
| **Navigation Flows** | Top 5 user journey paths |
| **Geographic breakdown** | Top countries by traffic share |

## 📊 Dashboard Sections

### 1. Overview
- 4 KPI cards (Views, Visitors, Bounce Rate, Session Duration)
- Daily traffic line chart
- Traffic sources donut chart
- Weekly sessions bar chart
- Device breakdown
- Conversion funnel

### 2. Traffic
- Full 30-day traffic trend (expanded view)
- Bounce rate per page (horizontal bar)
- Hourly traffic heatmap

### 3. Pages
- Full pages table with search + sort
- Views, unique visitors, bounce rate, avg time, page type

### 4. Behavior
- User navigation flows (entry → exit paths)
- Entry vs exit comparison chart
- Browser distribution
- Geographic traffic breakdown

### 5. Recommendations
- 4 prioritized action cards
- Impact vs Effort priority matrix
